## 先说说 Zipic 是个什么东西

简单讲，Zipic 是一款专为 macOS 设计的图片压缩软件，开发者是十里软件科技（济南）有限责任公司。它最大的特点是——**所有压缩操作都在本地完成**，不用把图片传到云端。对于处理商业素材、客户稿、个人照片的人来说，这一点挺重要，隐私和数据安全有保障。

它的核心能力包括：

- **批量压缩**：拖一整个文件夹进去，一次压完，不用一张张点
- **多格式支持**：JPEG、PNG、WebP、HEIC、GIF 这些基础格式免费版就能用；专业版还解锁 AVIF、TIFF、ICNS、SVG、PDF、JPEG XL、APNG 等高级格式
- **格式与尺寸转换**：批量把图片统一转成 WebP、AVIF，或者按指定宽高缩放
- **自动化监听**：专业版可以监控指定文件夹，来了新图自动压缩；还能监听剪贴板，截图后自动瘦身
- **拖到刘海压缩**：把图片直接拖到 MacBook 屏幕刘海区域就能压缩，用过就回不去的丝滑体验

按官方和第三方测评的数据，Zipic 能把图片体积缩小 **40%–70%**，而肉眼几乎看不出画质差异。我实测一张 5.2MB 的 PNG 压完变成 580KB，体积掉了 89%，发到博客上加载速度肉眼可见地变快。
